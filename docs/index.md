# Eclipse Kura™ Apache ActiveMQ Artemis MQTT broker

This official addon for Eclipse Kura™ provides the following components for configuring the bundled ActiveMQ-7 MQTT instance:

- [`org.eclipse.kura.broker.artemis.simple.mqtt`](./simple-artemis-mqtt-broker.md): component that allows to configure the broker using just a few options
- [`org.eclipse.kura.broker.artemis.xml`](./active-mq-artemis-broker.md): component that allows defining a detailed configuration through an XML description, unlocking the full capabilities of the ActiveMQ-7 MQTT instance
